## 🌐 部署方案

### 本地开发
```bash
# 快速启动
./start.sh

# 手动启动
pnpm install
cp .env.development .env
pnpm dev
```

### Vercel 部署（前端）
```bash
# 安装 Vercel CLI
npm i -g vercel

# 部署到生产环境
vercel --prod
```

**访问地址**: `https://your-app-name.vercel.app`

### 全栈部署（可选）
- **数据库配置**: MySQL 数据库连接
- **环境变量**: OAuth 服务配置
- **文件存储**: AWS S3 配置
